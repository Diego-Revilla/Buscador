------------------------------------------------------------------------
Este proyecto consistío en crear un buscador que sirviera para guaradar
notas creadas por un cliente y poder buscar en ellas coincidencias con 
alguna palabra en concreto.
Contiene 2 clases para su funcionamiento, la clase Buscador que almacena
las notas y puede hacer una bsuqueda general de una plabra y la clase
Nota que es la encargada de crear las notas que almacenarán nuestra info.
